---++ Usage

This plugin reads metadata stored in images stored as [[https://en.wikipedia.org/wiki/Exif][Exchangeable Image File Format]]. Typical information
contains the 

   * date
   * size
   * geometry 
   * geo location
   * name of the camera and its processing parameters
   * rotation orientation 
   * color space 
   * etc

Not every image contains all information, sometimes EXIF data has been stripped off from images, e.g. for privacy reasons.
The =%EXIF= macro takes the name of an image attached to a topic and formats it according to your. needs.

---++ Examples

You type:
<verbatim>
%EXIF{"exif.png" raw="on"}%
</verbatim>

You get:
%IF{"context ExifPluginEnabled"
   then="$percntEXIF{\"exif.png\" raw=\"on\"}$percnt"
   else="%RED%not installed%ENDCOLOR%"
}%

---++ Syntax

<verbatim>%EXIF{"attachment.png"
   web="web"
   topic="web.topic"
   format="..."
   raw="on/off"
   coord_format="..."
}%</verbatim>

| *Parameter* | *Description* | *Default* |
| "..." | attachment | |
| web="..." | web parameter | current web |
| topic="..." | topic from where the attachment is fetched | current topic |
| format="..." | format string, see below |  |
| raw="on/off" | boolean to display all of the available exif data in raw format | =off= |
| coord_format="..." | sprintf format of the coordinates | =%+.8f= |

The =format="..."= parameter may contain keys prefixed by a dollar sign, e.g. =$FileName= or =$FileType=.
Use the =raw="on"= parameter to see all of the available keys.
